Features
⚡

Event-Driven I/O

Single-threaded kqueue event loop handles thousands of concurrent connections with O(1) complexity.

🌐

HTTP/1.1 Protocol

Full request parsing (method, path, headers). Supports GET and HEAD with proper status codes.

📁

Static File Serving

Auto-detects 25+ MIME types. Serves HTML, CSS, JS, images, fonts, and more with correct headers.

🔒

Path Security

Built-in protection against directory traversal, absolute path injection, and null byte attacks.

📋

Directory Listing

Auto-generated, styled directory indexes with file sizes, dates, and parent directory navigation.

📝

Access Logging

Real-time request logging with timestamps, method, path, status codes, and response sizes.

Terminal — mini-httpd
